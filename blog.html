CSS HTML

CSS dapat ditambahkan ke dalam HTML dengan 3 cara:

1. Inline CSS

Contoh: <p style="color: red; font-size: 18px;">Teks dengan inline CSS</p> menjadi:

Teks dengan inline CSS

2. Internal CSS

Contoh:

<head>
  <style>
    .styled-text {
      color: blue;
      background-color: lightyellow;
      padding: 5px;
      border: 1px solid navy;
    }
  </style>
</head>
<body>
  <p class="styled-text">Teks dengan internal CSS</p>
</body>

Hasilnya akan menjadi seperti ini:

Teks dengan internal CSS

3. External CSS

Contoh:

<head>
  <link rel="stylesheet" href="styles.css">
</head>

Dimana file styles.css berisi:

body {
  font-family: Arial, sans-serif;
  margin: 20px;
}

h1 {
  color: navy;
  border-bottom: 2px solid #ccc;
}

.container {
  background-color: #f5f5f5;
  padding: 15px;
  border-radius: 5px;
}

CSS Selectors

Contoh selectors:

  • Element selector: p { color: red; }
  • ID selector: #header { background-color: black; }
  • Class selector: .highlight { background-color: yellow; }
  • Attribute selector: a[target="_blank"] { color: orange; }
  • Pseudo-class: button:hover { background-color: lightblue; }

HTML Links

Link ke halaman di website yang sama: <a href="index.html">Kembali ke Beranda</a> menjadi: Kembali ke Beranda

Link ke bagian di halaman yang sama: <a href="#css">Lihat CSS HTML</a> menjadi: Lihat CSS HTML

Link email: <a href="mailto:contoh@email.com">Kirim email</a> menjadi: Kirim email

Link telepon: <a href="tel:+6281234567890">Hubungi kami</a> menjadi: Hubungi kami

Link dengan judul: <a href="https://www.example.com" title="Kunjungi Example">Example.com</a> menjadi: Example.com (hover untuk melihat judul)

HTML Favicon

Favicon berguna untuk menggantikan icon di tab browser. Favicon biasanya berupa gambar kecil yang ditampilkan di sebelah judul halaman web.
